# Time Tracker

- A CLI utility that allow to track the time by starting, stoping and displaying the report of time tracks.


Usage: track [OPTIONS] <COMMAND>

Commands:
  - `start`   Start tracking time
  - `stop`    
  - `report`  
  - `help`    Print this message or the help of the given subcommand(s)

Options:
  - `-d, --db-dir <DB_DIR>`      Path to database file
  - `-l, --lockfile <LOCKFILE>`  Path to lockfile
  - `-h, --help`                 Print help
  - `-V, --version`              Print version
